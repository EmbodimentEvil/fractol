# Project for school_21.
## FRACTOL.
---
##### In this project I use OpenCl and create a program that displays 2D mathematical fractols.
---
## Compiling
Use `make` and run `./fractol [0-6]`.  
Follow the menu and use the keyboard and mouse for changes.
